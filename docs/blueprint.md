# **App Name**: Stream

## Core Features:

- Post Creation: Users can create and submit their own posts with text, images, and links.
- Content Feed: Display a real-time feed of posts from users the current user follows and trending content, sorted by recency.
- Search Functionality: Enable users to search for other users, posts, and topics using keywords.
- User Profiles: Allow users to create and customize their profiles, including profile picture, bio, and a display of their own posts.
- AI Post Suggestion: AI analyzes the user's recent activity, suggests topics or formats for the user to create a new post, functioning as a creative tool.
- Following System: Users can follow other users to see their posts in the content feed.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to convey trust and sociability.
- Background color: Very light blue (#F0F8FF), a desaturated variant of the primary color, to provide a clean and unobtrusive backdrop.
- Accent color: Pale purple (#BDB2FF) to highlight interactive elements and create visual interest.
- Body and headline font: 'PT Sans' for a modern and approachable feel.
- Use a set of minimalist icons for navigation and interactions. Ensure icons are clear, intuitive, and consistent throughout the app.
- Implement a clean, card-based layout for posts to enhance readability and visual organization. Use consistent spacing and padding.
- Incorporate subtle animations for loading new content, transitions between screens, and interactive feedback. Animations should be smooth and unobtrusive.
